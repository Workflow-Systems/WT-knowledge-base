# Сочетания клавиш

{% hint style="info" %}
Раскладка - это про символ на клавише и текущий язык ввода.
{% endhint %}

### **Редактирование**

| <p>Сочетания клавиш<br>[специальные контексты]</p>                 | Команды                                                                                                                                                            |
| ------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| <p><strong>Ctrl + &#x3C;</strong><br>[английская раскладка]</p>    | Закомментировать выделенный фрагмент. Чтобы раскомментировать фрагмент, достаточно поставить курсор в любое место комментария и нажать эту же комбинацию клавиш.   |
| <p><strong>Ctrl + Shift + /</strong><br>[английская раскладка]</p> | Закомментировать выделенный фрагмент.                                                                                                                              |
| <p><strong>Ctrl + Shift + \</strong><br>[английская раскладка]</p> | Раскомментировать выделенный фрагмент. Достаточно поставить курсор в любое место комментария и нажать эту же комбинацию клавиш.                                    |
| <p><strong>Ctrl + Shift + /</strong><br>[русская раскладка]</p>    | Комментирование выделенного фрагмента. Для снятия комментария с фрагмента достаточно поставить курсор в любое место комментария и нажать эту же комбинацию клавиш. |
| **Ctrl + Shift + С**                                               | Закомментировать или раскомментировать строку, на которой находится курсор.                                                                                        |
| <p><strong>Ctrl + ></strong><br>[английская раскладка]</p>         | Экранирование выделенного фрагмента символами `<![CDATA[]]>`                                                                                                       |
| <p><strong>Ctrl + .</strong><br>[русская раскладка]</p>            | Экранирование выделенного фрагмента символами `<![CDATA[]]>`                                                                                                       |
| **Ctrl + Q**                                                       | Форматировать документ.                                                                                                                                            |

### **Правка**

| <p>Сочетания клавиш<br>[специальные контексты]</p>          | Команды                                                                                                                                                                 |
| ----------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Ctrl + Space**                                            | Вызов окно автозаполнения                                                                                                                                               |
| <p><strong>Ctrl + click</strong><br>[на имени сущности]</p> | Переход к определению сущности в коде формы. Такой переход возможен к SqlQuery в серверном xml-файле или переход в xml-файл формы, указанный в команде FormShowCommand. |
| <p><strong>Ctrl + 1</strong><br>[на панели Numeric]</p>     | Выделение кода тэга. Если курсор стоит перед символом <, то выделится код от него и до конца закрывающего тэга. Иначе выделится код внешнего тэга.                      |
| <p><strong>Ctrl + 2</strong><br>[на панели Numeric]</p>     | Выделение значения тэга (константы или вложенного тэга) или значения атрибута, если курсор стоит на имени или значении атрибута тэга.                                   |
| <p><strong>Ctrl + 4</strong><br>[на панели Numeric]</p>     | Выделение кода внешнего тэга.                                                                                                                                           |
